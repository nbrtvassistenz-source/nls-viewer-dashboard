Einstellungen

Der Key wird nur lokal im Browser gespeichert und direkt an die YouTube-API gesendet. Quota: 1 Einheit pro Abfrage (alle 20s = ~4.300/Tag, Limit liegt bei 10.000).
Z.B. https://www.youtube.com/watch?v=xxxxxxxxxxx, https://youtu.be/xxxx, https://youtube.com/live/xxxx oder einfach die Video-ID.
Eine URL oder Video-ID pro Zeile. Spalten aus Excel/Google Sheets (mit Tabs getrennt) funktionieren auch — Duplikate und Leerzeilen werden automatisch ignoriert.

Dashboard teilen

Diese URL enthält deine komplette Konfiguration. Wer sie öffnet, bekommt das gleiche Setup automatisch angeboten.
⚠️ Mit eingeschlossenem Key kann jeder, der den Link hat, deine YouTube-API-Quota verbrauchen. Nur im internen Team teilen — oder den Key in der Google-Cloud-Console auf eine Domain beschränken.
